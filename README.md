# time


This is a date and time utility featuring some specific utilities

### Features
* **workingdays**
* **DateConverter**
* **DateTimeUtils**

### Adding time as a dependency

```scala
"uk.gov.hmrc" %% "time" % "[INSERT_VERSION]"
```

For Java 7 use a version <= 2.0.0
